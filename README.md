# Core_Group_Project_1

Overview

This is a front-end Human Resource Management System built using Vue.js.
All data is stored using localStorage and seeded from JSON files.
The system simulates real HR functions such as employee management, attendance, leave, and payroll.

Features

Employee Management – view employee details and search employee
Attendance Tracking – record hours worked
Leave Management – apply, approve/reject leave
Payroll Module
Automated salary calculation
Digital payslip modal
Downloadable PDF payslips (jsPDF)
Dashboard 
Salary Overview (charts.js)
Attendance
Leave Request
Mock Login (optional)


Technologies
